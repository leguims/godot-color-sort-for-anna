Liste et organisation pour Godot:
	OK - Jeton : cube de couleur
		OK - contient la repesentation d'un jeton

	- Pile : Colonne sur un plateau de jeu
		- Contient les regles de jeu subjectives :
			- Accepte un/des jeton(s) de couleur,
			- Donne un/des jeton(s) de couleur,
			- Est_terminé (colonne pleine mono-couleur),
			- Est_vide (aucun jeton dans la pile),
			- Est_pleine (aucune emplacement vide dans la pile)
		OK - Contient les caractéritstiques de la pile :
			OK - taille
			OK - Liste des jetons actuels
		OK - Encodage d'une pile:
			OK - "[0, 0, 0, 0]" = 4x'A' sur une pile de 4
			OK - "[0, 0, 0, 0, 32, 32]" = 4x'A' + 2x' ' sur une pile de 6
			OK - "[0, 1, 2, 32]" = 3x blocs et 1 case vide

	- Plateau : ensemble des piles de jeu
		OK - Associe plusieurs piles pour former le plateau
		- Contient les regles subjectives :
			- Liste des mouvements autorisés,
			- Est_terminé (toutes les colonnes sont terminées),
			- Est_bloqué (la liste des mouvements autorisés est vide)
		OK - Encodage de plateau:
			OK - "AABB.BBAA.    " signifie :
				OK - pile 1 : "AABB"
				OK - pile 2 : "BBAA"
				OK - pile 3 : vide (4 emplacements)
			OK - "ABAB.BABA.    . "
				OK - pile 1 : "ABAB"
				OK - pile 2 : "BABA"
				OK - pile 3 : vide (4 emplacements)
				OK - pile 4 : vide (1 emplacement)

	- Menu :
		OK - Page d'accueil
		- Liens entre les plateaux
		OK - Ligne de saisie pour générer un plateau à résoudre.
		- Lien vers les Crédits (GODOT, musique, effet sonore)
